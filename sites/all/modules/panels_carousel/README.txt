Description
-----------
This module provides a configurable "carousel" Panel style that can be used by
the Mini Panels module.


Dependencies
------------
* jCarousel (http://drupal.org/project/jcarousel)


Installation
------------
1) Place this module directory in your modules folder (this will usually be
"sites/all/modules/").

2) Enable the module.
